- PROJECT ONE -


Digital Product Passport

A digital identifier designed to trace the entire life cycle of a product. In the case of a garment, this digital ID is embodied by the QR code imprinted on the care label of the apparel. Avery Dennison’s Digital Care Label takes the traditional care label one step further, providing information digitally, and helping brands prepare for coming legislation changes.

- PROJECT TWO -


Weave - Design System

We built a design system Weave. And we linked the design system with Storybook, which enhances the development and maintenance of user interfaces. Weave was originally built in Sketch.

  • Style guidelines:
    A set of rules that define the visual style of the user interface, including typography, colors, and spacing.
  • Component library:
    A collection of pre-built components that can be used to create user interfaces.
  • Documentation:
    A comprehensive set of documentation that explains how to use the design system, including style guidelines, component usage, and best practices.
