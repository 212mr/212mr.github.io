SMARTDELIVERY

UX / UI Design Case

This is a mobile app used to summarize online shopping package delivery statuses by auto-detecting shopping emails. This is an individual project initializing from idea exploration through to user experience research. After many rounds of user tests with wireframes, interface prototypes were designed and implemented.
