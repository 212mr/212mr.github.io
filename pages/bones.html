Frontend Development

Frontend Development

A lot of materials were required to initialize the build for this website. There were many challenged in creating a product list page, however I decided to use PHP and MySQL to store product metadata. I also used PHP to give each product an ID. I took photos for all these flowers, and you can see them in my portfolio photography collection.

  • HTML5 / CSS3
  • PHP
  • JavaScript / JQuery

Backend Development

It wasn't too challenging to learn PHP and MySQL at the beginning, but I did still have to Google a few things! An administration page was created to get/add data of product details. If more time permitted, I may have approached this project differently by building new frameworks (React or Angular or Vue), however, the final product is still a joy to use. I gave my heart to this project; I hope you enjoy my flower shop!
