极省海运 运费细则

系统自动计算结果基于如下参数：

* 空运：3-7天 到达

* 运费单价： $1.2 / 100g （即 每100克1.2美元）。

* 不足100g的向上取整收费，如：120g 收费重量为200g。

* 实际重量和抛货重量取比较重的那个：
抛货重量（公斤）计算公式：长(cm) x 宽(cm) x 高(cm) / 5000。

* 如果单个包裹收费重量大于2kg, 价格自动变为9折 。



例如：长33厘米 x 宽33厘米 x 高33厘米 / 5000 = 7.1874 公斤。
运费：7187克 取算7200克；7200克 x 1.2美元 / 100克 x 90% (打九折) = 77.76 美元。


* 包裹中如有片状面膜，每单收取3刀额外费用，重量不限。

* 带有内置电池的产品，每单收取3刀额外费用，重量不限。

* 包裹单边长超过1.2米，收取超长费$180。

系统自动计算结果基于如下参数：

* 本计算公式只用于 八大开放品名。

* 海运：3 - 4周 达到。

* 单价：$5/kg， 单个包裹重量低于3kg的，按照3kg收费。

* 不足1kg的，向上取整收费，如：4.3kg收费重量为5kg。

* 实际重量和抛货重量取比较重的那个：
抛货重量（公斤）计算公式：长(cm) x 宽(cm) x 高(cm) / 6000。



例如：长33厘米 x 宽33厘米 x 高33厘米 / 6000 = 5.9895 公斤。
运费：5.9895 公斤，取整数6公斤； 6公斤 x $5/kg = $30 美元。




* 超长超重费 （如有疑问请先咨询客服）：

1）包裹单边长超过110cm 加收$20/箱；长 + 2 x（宽 + 高）大于300 加收$100/箱。

2）不可分拆的单个物品实重超过30KG但小于50kg需加收$20/箱；大于50kg且不超过68KG需加收$100/箱。

3）没有用任何东西包装或者用保鲜膜，布料，金属性的（如不锈钢）作为外包装或外包装是圆柱形需加收$20/箱。不接受木箱包装。
